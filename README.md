# Neighborhood Library Application
## Overview
 This app was created by Beatriz Jardim for the Java Focus Class workshop. Anyone can check out a book, they just enter their name and the application will track who checked it out.
## Features
 - It has 1 main menu with while and switch case where you can choose between seeing all available books or seeing all checked out books.
 - Sub-menus where you can choose to check out a book or check in a book.
 - Welcome and goodbye messages.
 - loadInventory() with 20 queer books
## Tech Used / Dependencies
 - This is a Java App, I used IntelliJ for IDE 
 - Used Claude AI as learning assistant for explanations and for guidance while refactoring - all code was written independently
## Running the project
 1. Clone the repository
 2. Open in IntelliJ IDEA
 3. Navigate to LibraryProgram.java
 4. Click the green Run button
